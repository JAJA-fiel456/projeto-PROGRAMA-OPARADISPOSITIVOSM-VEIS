# 🛒 eCompra

Um aplicativo simples e funcional para cadastrar e gerenciar produtos, desenvolvido com **React Native** e **Expo**.

---

## ✨ Sobre o Projeto

O **eCompra** é um gerenciador de produtos que permite adicionar, visualizar e excluir itens de forma prática. Ideal para pequenos negócios, feiras ou como base para um app de e-commerce mobile.

### Funcionalidades

- ✅ Cadastro de produtos com título, descrição, preço, telefone, quantidade e imagem
- ✅ Listagem de todos os produtos cadastrados
- ✅ Armazenamento local utilizando **SQLite** (`expo-sqlite`)
- ✅ Interface intuitiva e responsiva
- ✅ Suporte a dispositivos móveis (Android e iOS)

---

## 🛠️ Tecnologias Utilizadas

- **React Native**
- **Expo**
- **expo-sqlite** (banco de dados local)
- **React Hooks** (`useState`, `useEffect`)
- **JavaScript**
